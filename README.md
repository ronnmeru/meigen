### テーマを選んだ理由
本当に辛いとき自分は偉人の名言が思い浮かんだ。それが支えになったので、辛いときに自分奮い立たせてくれる名言を共有できるサイトを製作しようと思った。

老若男女全員楽しめるが主なターゲットは自分と同じような世代の男性。挑戦するときの支えになると思う。

### 主な利用シーン
挑戦するとき、失敗したとき、奮い立たせるとき。
